# MTG_Cube
MTG Cube draft simulator for quick play with friends

## setup dev environment
### client
development
```
cd client
npm i
npm run dev
```
### server
development
```
cd server
npm i
npm run dev
```
tests
```
cd server
npm run test
```

### external dependencies
client
- react-router-dom
- bootstrap
- react-bootstrap
- socket.io-client

server
- npm i joi@13.1.0
- npm i express
- npm i cors
- npm i body-parser
- npm i fs
- npm i path
- npm i socket.io
